# 30 Circles

A creativity challenge game inspired by the classic 30 Circles exercise. Turn as many blank circles as possible into recognizable objects in 3 minutes.

**[Play it here →](https://martuuflis.github.io/30-circles/)**

## How it works

1. Tap **Let's start!** to begin the 3-minute timer
2. Draw anything inside each circle — a face, a planet, a clock, whatever comes to mind
3. Tap **Next** to move to the next circle
4. When time runs out (or you finish all 30), see how many you completed

## Built with

- React 18 (via CDN, no build step)
- Canvas API for freehand drawing
- Vanilla CSS animations
